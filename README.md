# Web-project-01

## Portfolio Website for Prasad Mohapatra

This project is a beginner-friendly portfolio website built with only HTML, CSS, and JavaScript.

### Folder structure

```
Web-project-01/
├─ index.html
├─ css/
│  └─ styles.css
├─ js/
│  └─ script.js
└─ README.md
```

### How to run locally

1. Open the project folder in VS Code.
2. Open `index.html` directly in your browser.
3. Or right-click `index.html` and choose "Open with Live Server" if you have that extension.

### How to deploy for free

Use any of these services:

- **GitHub Pages**: Push the repository to GitHub, then enable Pages in repository settings from the `main` branch and root folder.
- **Netlify**: Drag and drop the project folder into Netlify Deploy, or connect the GitHub repo and set the build command empty.
- **Vercel**: Import the GitHub repository, then deploy with project settings using the `root` folder. No build step is needed.

### Notes

- This site is responsive and uses a mobile-friendly navigation menu.
- The contact form shows a friendly message without sending real emails, so it is safe for a beginner portfolio.
